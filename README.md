# IsleBloom: Island Plant Competition Simulator

## Overview
**IsleBloom** is an interactive ecological simulation web application that models the survival, reproduction, and interspecific competition dynamics of plant populations in a limited island environment.

Built as a university coursework project using AI-assisted development (Arena.ai), it allows users to design custom island terrains and plant species parameters to explore "survival of the fittest" principles in ecology and computational biology.

**Live Demo**  
👉 [https://lirun478.github.io/IsleBloom/](https://lirun478.github.io/IsleBloom/)

## Features
- 🗺️ **Island Editor**: Freely draw Water, Beach, Grass, and Forest terrains with brush tools
- 🌱 **Custom Plant Species**: Add unlimited species with full parameter control
- ⚙️ **Rich Ecological Parameters**: Lifespan, juvenile period, pollination range, seed dispersal, mutation rate, intra- and inter-specific competition
- 📊 **Real-time Simulation & Statistics**: Live population counters, individual plant tooltips, competition effects, and population change charts
- 🌍 **Bilingual Interface**: English + Chinese
- 📱 **Fully responsive** single-file HTML/JS web app — no installation required

## How to Use
1. Open the [live demo](https://lirun478.github.io/IsleBloom/)
2. Click **START** to enter the Island Editor
3. Design your island terrain and customize plant species
4. Click **Start Simulation** and observe the ecological dynamics in real time
5. Click **Intro & Instructions** for detailed guidance

## Technical Stack
- Pure HTML5 + Tailwind CSS + Chart.js + Vanilla JavaScript  
- Grid-based spatial simulation with per-round life cycles, mutation, and density-dependent competition  
- Runs entirely in the browser (no backend required)
